# Frontend 

-----------------
## Commencer
###  
1. Installer les dépendences  
````shell
$ npm install
````

3. Lancer le serveur à l'aide de la commande " start " du package.json ou ↓  
````shell
$ ng serve
````
4. Le frontend est lancée sur: http://localhost:4200/
